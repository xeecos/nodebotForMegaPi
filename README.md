# NodeBot For MegaPi & Raspberry Pi
 * git clone https://github.com/xeecos/nodebotForMegaPi on your raspberry pi
 * cd nodebotForMegaPi
 * npm install megapi  express
 * node app.js
 * open http://raspberrypi.local:3000 on your browser
